### 1. 大势择时模型

#### ✅ 多因子打分系统 ([strategies/timing/macro_scorecard.py](strategies/timing/macro_scorecard.py))

**核心功能**：
1. **宏观状态得分**：基于MS模型的状态概率
   - 正常增长（Regime_1）：100分
   - 过热（Regime_2）：50分
   - 衰退（Regime_3）：0分
   - 滞胀（Regime_4）：25分

2. **流动性得分**：M1-M2剪刀差、社融增速
   - M1增速、M2增速、剪刀差
   - 社融增速（可选）
   - 滚动窗口标准化（252天）

3. **估值得分**：股债风险溢价（ERP）
   - ERP = 股票盈利收益率 - 无风险收益率
   - 滚动窗口标准化

4. **外资情绪得分**：北向资金流、VIX指数
   - 北向资金流（主要）
   - VIX指数（反向）
   - 资金流预测（可选）

5. **综合得分**：加权平均
   - 宏观状态权重：30%
   - 流动性权重：25%
   - 估值权重：25%
   - 情绪权重：20%

**API示例**：
```python
from strategies.timing.macro_scorecard import MacroScorecard

# 初始化打分系统
scorecard = MacroScorecard(
    macro_weight=0.30,
    liquidity_weight=0.25,
    valuation_weight=0.25,
    sentiment_weight=0.20
)

# 计算各类得分
macro_score = scorecard.calculate_macro_score(regime_probs)
liquidity_score = scorecard.calculate_liquidity_score(m1_data, m2_data, social_financing)
valuation_score = scorecard.calculate_valuation_score(stock_price, bond_yield)
sentiment_score = scorecard.calculate_sentiment_score(northbound_flow, vix)

# 计算综合得分
composite_score = scorecard.calculate_composite_score(
    macro_score, liquidity_score, valuation_score, sentiment_score
)
```

**输出**：
- 各类因子得分（0-100）
- 综合得分（0-100）
- 得分摘要

#### ✅ 信号生成器 ([strategies/timing/signal_generator.py](strategies/timing/signal_generator.py))

**核心功能**：
1. **得分转仓位**：将综合得分转换为仓位建议
   - 得分 < 20：0%仓位
   - 得分 20-35：25%仓位
   - 得分 35-50：50%仓位
   - 得分 50-65：75%仓位
   - 得分 > 80：100%仓位

2. **风险调整**：根据风险指数动态缩减仓位
   - 高风险（>70）：缩减至50%
   - 中等风险（40-70）：缩减至75%
   - 低风险（<40）：保持不变

3. **交易信号生成**：
   - 买入信号（+1）：仓位增加超过5%
   - 卖出信号（-1）：仓位减少超过5%
   - 持有信号（0）：变化小于5%

4. **信号平滑**：多数投票法，避免频繁交易

**API示例**：
```python
from strategies.timing.signal_generator import SignalGenerator

# 初始化信号生成器
generator = SignalGenerator(
    score_thresholds={'very_low': 20, 'low': 35, 'medium': 50, 'high': 65, 'very_high': 80},
    position_levels={'very_low': 0.0, 'low': 0.25, 'medium': 0.50, 'high': 0.75, 'very_high': 1.0},
    risk_adjustment=True,
    signal_smoothing=True
)

# 生成所有信号
results = generator.generate_all(
    composite_score=composite_score,
    risk_index=risk_index,
    signal_threshold=0.05
)

# 结果包含：
# - position: 原始仓位
# - risk_adjusted_position: 风险调整后仓位
# - signals: 交易信号
# - orders: 交易指令
```

**输出**：
- 仓位序列（0.0-1.0）
- 交易信号序列（-1, 0, 1）
- 交易指令DataFrame

### 2. Backtrader回测框架

#### ✅ 自定义数据源 ([backtest/aligner/macro_data_feed.py](backtest/aligner/macro_data_feed.py))

**核心功能**：
1. **集成日历对齐器**：确保不会使用未来数据
2. **支持宏观数据**：在价格数据基础上添加宏观数据字段
3. **混频数据处理**：自动前向填充缺失的宏观数据
4. **多资产支持**：同时加载多个资产的数据

**数据字段**：
```python
# 基础字段（继承自PandasData）
datetime, open, high, low, close, volume

# 宏观数据字段（新增）
macro_score          # 综合宏观得分
liquidity_score      # 流动性得分
valuation_score      # 估值得分
sentiment_score      # 情绪得分
risk_index           # 风险指数
regime_prob_1        # 区制1概率
regime_prob_2        # 区制2概率
regime_prob_3        # 区制3概率
regime_prob_4        # 区制4概率
```

**API示例**：
```python
from backtest.aligner.macro_data_feed import MacroDataFactory

# 创建数据工厂

```

#### ✅ Backtrader策略类 ([backtest/strategies/macro_strategy.py](backtest/strategies/macro_strategy.py))

**核心功能**：
1. **基于宏观因子的交易逻辑**：
   - 根据综合得分确定目标仓位
   - 根据风险指数动态调整仓位
   - 避免频繁交易（设置仓位变化阈值）

2. **绩效跟踪**：
   - 权益曲线记录
   - 回撤跟踪
   - 交易信号统计

3. **基准策略**：买入持有（BuyAndHoldStrategy）

**策略参数**：
```python
params = (
    # 宏观得分权重
    ('macro_weight', 0.30),
    ('liquidity_weight', 0.25),
    ('valuation_weight', 0.25),
    ('sentiment_weight', 0.20),

    # 信号参数
    ('score_threshold_low', 35),
    ('score_threshold_high', 65),
    ('position_threshold', 0.05),

    # 风险管理
    ('risk_adjustment', True),
    ('risk_threshold_high', 70),
    ('max_position', 1.0),
    ('min_position', 0.0),
)
```

**API示例**：
```python
import backtrader as bt
from backtest.strategies.macro_strategy import MacroStrategy

# 创建Cerebro引擎
cerebro = bt.Cerebro()

# 添加策略
cerebro.addstrategy(
    MacroStrategy,
    macro_weight=0.30,
    risk_adjustment=True
)

# 运行回测
results = cerebro.run()
```
