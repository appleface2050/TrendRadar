### 4. 核心基础设施

#### ✅ 日历对齐器 ([calendar_aligner.py](utils/calendar_aligner.py)) - **核心模块！**

这是整个回测系统的**生命线**，确保回测真实性。

**核心功能**：
1. **获取指定日期实际可观测的宏观数据值**
   - 杜绝未来函数（look-ahead bias）
   - 基于发布日历，而非数据日期

2. **支持混频数据**
   - 日度、周度、月度、季度数据统一处理
   - 自动选择最新的已发布数据

3. **中美两国数据统一接口**
   - 同一接口访问两国数据
   - 自动处理数据源切换

**API使用示例**：
```python
from utils.calendar_aligner import get_available_data, get_indicator_value

# 获取2024-01-15可观测的所有美国宏观数据
data = get_available_data('2024-01-15', country='US')

# 获取单个指标值（已考虑发布延迟）
gdp_value = get_indicator_value('2024-01-15', 'GDP', 'US')
```

**验证机制**：
- `validate_data_integrity()` - 验证数据完整性
- `get_release_lag_stats()` - 获取发布延迟统计

## ⚠️ 重要注意事项

### 1. 日历对齐的重要性

**错误示例（引入未来函数）**：
```python
# ❌ 错误：直接使用数据日期
gdp = df.loc['2024-01-15']['GDP']
# 问题：2024-01-15的GDP数据可能要到2024-04-30才发布！
```

**正确示例（使用日历对齐器）**：
```python
# ✅ 正确：使用日历对齐器
gdp = get_indicator_value('2024-01-15', 'GDP', 'US')
# 只获取在2024-01-15这一天实际已经发布的数据
```

### 2. API密钥配置

**配置方式**：编辑项目根目录的 `confidential.json`

```json
{
    "FRED_API_Key": "your_fred_api_key",
    "TUSHARE_DataApi__token": "your_tushare_token",
    "TUSHARE_DataApi__http_url": "http://1w1a.xiximiao.com/dataapi"
}
```

- 注册地址：https://tushare.pro/register
- 配置位置：`confidential.json` 中的 `TUSHARE_DataApi__token` 字段
- 特殊账号需要配置 `TUSHARE_DataApi__http_url` 字段
