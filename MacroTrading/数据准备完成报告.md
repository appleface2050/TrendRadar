## 📝 数据格式说明

### 标准格式（所有CSV文件）

```csv
date,value1,value2,...
2016-01-04,100.5,200.3,...
2016-01-05,101.2,201.5,...
...
```

### 价格数据格式（OHLCV）

```csv
date,open,high,low,close,volume
2016-01-04,3725.89,3725.89,3629.45,3636.84,123456789
...
```

### 日期格式要求

- ✅ 推荐格式：`YYYY-MM-DD` (如：2016-01-04)
- ✅ 兼容格式：`YYYYMMDD` (如：20160104)
- ❌ 不支持：其他格式

## 📞 技术支持

### 常见问题

**Q1: 数据文件位置不对？**
A: 所有数据应保存在 `MacroTrading/data/` 目录下，按照三级结构组织：
- `data/processed/china/` - 中国数据
- `data/processed/us/` - 美国数据
- `data/processed/global/` - 全球数据

**Q2: 如何验证数据格式？**
A: 运行数据验证脚本：
```bash
python scripts/validate_data.py
```

**Q3: Tushare Token 失效怎么办？**
A: 参考 `tushare_example.py` 的正确使用方式：
```python
import tushare as ts
pro = ts.pro_api()
pro._DataApi__token = 'your_token_here'
pro._DataApi__http_url = 'http://1w1a.xiximiao.com/dataapi'
```
