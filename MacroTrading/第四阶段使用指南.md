## 📝 数据生成示例

如果您的数据格式不同，可以使用以下代码转换：

```python
import pandas as pd

# 示例：生成宏观得分数据
from strategies.timing.macro_scorecard import MacroScorecard

# 1. 准备数据
regime_probs = pd.read_csv('regime_probs.csv', index_col='date', parse_dates=['date'])
m1_data = pd.read_csv('m1.csv', index_col='date', parse_dates=['date'])['value']
m2_data = pd.read_csv('m2.csv', index_col='date', parse_dates=['date'])['value']
stock_price = pd.read_csv('stock_price.csv', index_col='date', parse_dates=['date'])['close']
bond_yield = pd.read_csv('bond_yield.csv', index_col='date', parse_dates=['date'])['yield']
northbound_flow = pd.read_csv('northbound.csv', index_col='date', parse_dates=['date'])['flow']

# 2. 计算得分
scorecard = MacroScorecard()
macro_score = scorecard.calculate_macro_score(regime_probs)
liquidity_score = scorecard.calculate_liquidity_score(m1_data, m2_data)
valuation_score = scorecard.calculate_valuation_score(stock_price, bond_yield)
sentiment_score = scorecard.calculate_sentiment_score(northbound_flow)
composite_score = scorecard.calculate_composite_score(
    macro_score, liquidity_score, valuation_score, sentiment_score
)

# 3. 合并并导出
df = pd.DataFrame({
    'macro_score': macro_score,
    'liquidity_score': liquidity_score,
    'valuation_score': valuation_score,
    'sentiment_score': sentiment_score,
    'composite_score': composite_score
})

df.to_csv('data/derived/indicators/macro_scores.csv', encoding='utf-8-sig')
print("✅ 宏观得分数据已生成！")
```

## 📚 模块详解

### 1. 多因子打分系统

**功能**：将4类宏观因子标准化为0-100得分

**输入**：
- 区制概率（MS模型输出）
- M1、M2数据
- 股票价格、债券收益率
- 北向资金流、VIX

**输出**：
- 各类因子得分（0-100）
- 综合得分（0-100）

### 2. 信号生成器

**功能**：将综合得分转换为交易信号

**输入**：
- 综合得分序列
- 风险指数（可选）

**输出**：
- 仓位建议（0-100%）
- 交易信号（-1, 0, 1）
- 交易指令

### 3. Backtrader策略

**功能**：在Backtrader框架中执行宏观策略

**特点**：
- 集成日历对齐器
- 自动风险管理
- 完整绩效跟踪

### 4. 行业轮动

**功能**：基于宏观状态选择行业

**方法**：
1. 根据区制筛选行业池
2. 根据因子变化二次筛选
3. 动量确认
4. 计算权重

### 5. Streamlit仪表盘

**功能**：实时监控所有关键指标

**优势**：
- 无需编程
- 可视化友好
- 实时更新

## ⚠️ 常见问题

### Q1: 缺少某些数据文件怎么办？

**A**: 您可以：
1. 使用前三个阶段生成的数据
2. 从公开数据源下载
3. 使用模拟数据进行测试

### Q2: 如何自定义因子权重？

**A**: 在仪表盘侧边栏调整，或在代码中修改：
```python
scorecard = MacroScorecard(
    macro_weight=0.40,      # 提高宏观权重
    liquidity_weight=0.20,  # 降低流动性权重
    valuation_weight=0.25,
    sentiment_weight=0.15
)
```

### Q3: 如何添加自定义行业？

**A**: 在行业选择器中修改敏感度矩阵：
```python
sensitivities = {
    '您的行业': {
        'growth': 0.6,
        'inflation': -0.3,
        'liquidity': 0.5
    }
}
```

### Q4: 回测内存不足怎么办？

**A**:
1. 使用数据采样：`df.iloc[::10]`（每10个样本取1个）
2. 分段回测：按年份分别回测
3. 减少数据范围
