### 2. 资金流三层驱动模型 ([models/flow/flow_driver_model.py](models/flow/flow_driver_model.py))

#### ✅ 模型架构

**Layer 1: 全球层（Global）**
- VIX指数：全球风险偏好
- 美元指数（DXY）：美元强弱

**Layer 2: 相对吸引力层（Attraction）**
- 中美实际利差：资金套利空间
- 增长预期差（预留接口）

**Layer 3: 市场微观层（Micro）**
- AH溢价：A股相对H股的溢价
- USDCNH期权隐含波动率（预留接口）

#### ✅ 核心功能
1. **特征工程**：
   - 滞后特征（lag features）
   - 滚动窗口统计（均值、标准差、最大值、最小值）

2. **模型选择**：
   - OLS回归（基准）
   - Ridge回归（L2正则化）
   - Lasso回归（L1正则化）
   - 随机森林（非线性）

3. **分析输出**：
   - 各层重要性得分
   - 特征重要性排序
   - 模型性能指标（R²、RMSE）

#### API示例
```python
from models.flow.flow_driver_model import FlowDriverModel

# 初始化模型
model = FlowDriverModel(method='ols')

# 拟合模型
model.fit(data_dict, flow_data)

# 输出摘要
model.summary()

# 获取各层贡献
contribution = model.get_layer_contribution(data_dict)
```

#### 输出示例
```
各层重要性：
  全球层              : 45.23%
  相对吸引力层        : 32.15%
  市场微观层          : 22.62%

特征重要性：
  vix                 (global      ): 0.3521
  rate_diff           (attraction  ): 0.2845
  dxy                 (global      ): 0.1532
  ah_premium          (micro       ): 0.1203
```

### 3. 资金流Nowcasting模型 ([models/flow/flow_nowcasting.py](models/flow/flow_nowcasting.py))

#### ✅ 核心功能
1. **短期预测**：预测未来1-5日的资金流方向和规模
2. **方向预测**：预测资金流入/流出方向
3. **不确定性量化**：提供预测区间（95%置信度）

#### 特征工程
1. **滞后特征**：1-5阶滞后
2. **滚动特征**：5日、10日、20日滚动统计
3. **外部特征**：
   - VIX、DXY等全球因素
   - 中美利差、AH溢价等本地因素

#### 模型选择
- **随机森林**（RF）：非线性，特征重要性
- **梯度提升**（GBM）：更强的非线性能力
- **岭回归**（Ridge）：线性基准

#### API示例
```python
from models.flow.flow_nowcasting import FlowNowcastingModel

# 初始化模型（预测未来5天）
model = FlowNowcastingModel(horizon_days=5, method='rf')

# 训练模型
model.fit(flow_data, external_data=data_dict)

# 预测
result = model.predict(recent_data, external_data=data_dict)

print(f"预测方向：{result['direction']}")
print(f"预测值：{result['prediction']:.2f}")
print(f"预测区间：[{result['prediction_lower']:.2f}, {result['prediction_upper']:.2f}]")
```

#### 性能指标
```
模型性能：
  训练集 R²: 0.6234
  测试集 R²: 0.5123
  方向准确率: 62.45%
```

#### ✅ 指标体系

**1. 市场情绪指标**
- 波动率偏度（基于收益率）
- VIX MA比率（短期/长期）
- 新高新低比

**2. 流动性压力指标**
- 流动性利差（长期-短期利率）
- TED spread（3M LIBOR - 3M国债）
- 信用利差（公司债 - 国债）

**3. 杠杆水平**
- 融资交易占比（融资余额/市值）

**4. 外资行为**
- 北向资金Z-score
- 资金流波动率

### 6. 复合风险预警指数 ([models/risk/composite_risk_index.py](models/risk/composite_risk_index.py))

#### ✅ 核心功能
1. **整合指标**：综合市场风险指标和宏观风险指标
2. **机器学习**：XGBoost/LightGBM动态赋权
3. **预测回撤**：预测未来5日/20日市场回撤
4. **风险指数**：输出0-100的综合风险指数

#### 工作流程
```
1. 数据收集
   ├─ 市场风险指标（15+指标）
   └─ 宏观风险指标（10+指标）

2. 特征工程
   ├─ 滞后特征（1-3阶）
   └─ 滚动特征（5日、10日）

3. 模型训练
   ├─ XGBoost / LightGBM
   └─ 目标：未来N天回撤

4. 风险指数
   └─ 转换为0-100综合评分
```

#### API示例
```python
from models.risk.composite_risk_index import CompositeRiskIndex

# 初始化
cri = CompositeRiskIndex(model_type='xgboost', forecast_horizon=5)

# 训练模型
cri.fit(market_indicators, macro_indicators, price_data)

# 预测风险指数
risk_index = cri.predict(market_indicators, macro_indicators)

# 获取当前风险等级
current_risk = cri.get_current_risk_level(market_indicators, macro_indicators)
print(f"当前风险：{current_risk['risk_level']}")
print(f"风险指数：{current_risk['risk_index']:.2f}")
print(f"风险等级：{current_risk['risk_color']}")

# 导出风险指数
cri.export_risk_index(market_indicators, macro_indicators)
```

## ⚠️ 重要注意事项

### 1. 数据质量
- **垃圾进垃圾出（GIGO）**：数据质量决定模型上限
- **数据更新**：资金流数据需每日更新
- **多源验证**：建议多源交叉验证

### 2. 模型选择
- **资金流驱动模型**：
  - OLS：快速基准
  - RF：非线性能力强
- **Nowcasting模型**：
  - RF：特征重要性可解释
  - GBM：预测精度更高
- **风险指数**：
  - XGBoost：综合性能最佳
  - LightGBM：训练速度快

### 3. 参数调优
- **预测天数**：1-5日（短期），适合交易
- **滚动窗口**：5/10/20日，根据数据频率调整
- **风险阈值**：70（高风险），可根据实际需求调整

### 4. 样本外验证
- **重要**：严格避免未来函数
- **建议**：保留最近20%数据作为测试集
- **回测**：使用滚动窗口验证
