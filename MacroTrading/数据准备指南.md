## 📊 1. 价格数据（必需）

### 1.1 市场指数数据

**用途**：作为交易标的，计算收益率

**所需字段**：
```csv
date,open,high,low,close,volume

2016-01-05,3635.84,3677.56,3628.23,3664.12,987654321
...
```

**数据来源**：
- **Tushare**：`pro.index_daily(ts_code='000300.SH')` - 沪深300
- **Akshare**：`ak.stock_zh_index_daily(symbol='sh000300')`
- **Wind/Choice**：本地导出

**具体要求**：
- ✅ 时间范围：2016-01-01 至 2024-12-31
- ✅ 频率：日度
- ✅ 字段：date, open, high, low, close, volume
- ✅ 格式：CSV，UTF-8编码

**文件保存路径**：
```
data/processed/china/hs300.csv
```

## 💰 2. 宏观数据（必需）

### 2.1 货币供应量数据（M1、M2）

**用途**：计算流动性得分

**所需字段**：
```csv
date,m1,m2
2016-01-01,420000.5,1400000.8
2016-02-01,421000.3,1405000.2
...
```

**数据来源**：
- **Tushare**：
  ```python
  pro.cn_m(method='m2')  # M2数据
  pro.cn_m(method='m1')  # M1数据
  ```
- **中国人民银行官网**：统计数据 → 货币统计

**具体要求**：
- ✅ 时间范围：2016-01-01 至 2024-12-31
- ✅ 频率：月度（会自动扩展到日度）
- ✅ 字段：date, m1（亿元）, m2（亿元）
- ✅ 注意：使用**同比增速**或**原值**都可以

**文件保存路径**：
```
data/processed/china/m1_m2.csv
```

### 2.2 债券收益率数据

**用途**：计算估值得分（股债风险溢价）

**所需字段**：
```csv
date,yield
2016-01-04,2.85
2016-01-05,2.87
...
```

**数据来源**：
- **Tushare**：`pro.cn_bnd(ts_code='020005.IB')` - 10年期国债
- **中债登**：http://www.chinabond.com.cn/
- **FRED**（如果用美债）：`GS10` - 10年期国债收益率

**具体要求**：
- ✅ 时间范围：2016-01-01 至 2024-12-31
- ✅ 频率：日度
- ✅ 字段：date, yield（%）
- ✅ 期限：10年期国债

**文件保存路径**：
```
data/processed/china/bond_yield_10y.csv
```

### 2.3 北向资金流数据

**用途**：计算情绪得分

**所需字段**：
```csv
date,northbound_flow
2016-12-05,1234567890
2016-12-06,-234567890
...
```

**数据来源**：
- **Tushare**：
  ```python
  pro.moneyflow_hsgt(hk_type='北向')  # 沪股通、深股通数据
  ```
- **AKShare**：`ak.stock_em_hsgt_north_net_flow_in()`

**具体要求**：
- ✅ 时间范围：2016-01-01 或 2017-03-17（沪深通开通）至 2024-12-31
- ✅ 频率：日度
- ✅ 字段：date, northbound_flow（元）
- ✅ 说明：正值表示流入，负值表示流出

**文件保存路径**：
```
data/processed/china/northbound_flow.csv
```
