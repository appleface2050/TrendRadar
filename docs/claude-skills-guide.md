## 📚 文档处理类

### 1. DOCX - Word 文档处理

**技能名称**: `docx`

**功能描述**:
全面的 Word 文档创建、编辑和分析工具。支持跟踪更改、添加注释、保留格式和文本提取。

**主要特性**:
- 创建专业的 Word 文档
- 编辑现有文档内容
- 处理跟踪修订（Track Changes）
- 添加和管理注释
- 保留原有格式
- 提取文档中的文本内容

**使用场景**:
- 创建商业报告和合同
- 编辑和审阅文档
- 批量处理 Word 文档
- 提取文档内容用于分析

**示例用法**:
```
- "创建一份年度销售报告的 Word 文档"
- "编辑这个合同，添加修订跟踪"
- "从 report.docx 提取所有表格数据"
- "在这个文档中添加注释说明"
```

## 🚀 如何使用 Skills

### 在 Claude Code 中使用

1. **已启用的技能**:
   - ✅ `document-skills` (包含 docx, pdf, pptx, xlsx)
   - ✅ `example-skills` (包含所有其他技能)

2. **直接对话使用**:
   直接在对话中提及需要完成的任务，Claude 会自动识别并使用相应的技能。

   ```
   示例：
   - "使用 PDF 技能从 report.pdf 提取数据"
   - "创建一个关于销售数据的 Excel 报表"
   - "用 PowerPoint 技能制作产品演示"
   - "设计一个现代化的登录页面"
   ```

3. **显式指定技能**:
   你可以明确指定要使用的技能：
   ```
   - "使用 Web 测试技能验证我的应用"
   - "用算法艺术技能创建一个生成艺术"
   ```

### 安装新技能

如果需要安装额外的技能：

```bash
# 在 Claude Code CLI 中执行
/plugin install <skill-name>@anthropic-agent-skills
```

### 创建自定义技能

1. 创建技能文件夹：
   ```bash
   mkdir my-custom-skill
   cd my-custom-skill
   ```

2. 创建 `SKILL.md` 文件：
   ```markdown
   ---
   name: my-custom-skill
   description: 这个技能用于[描述使用场景]
   ---

   # 我的技能

   ## 使用指南
   [详细的指令]
   ```

3. 打包并上传：
   ```bash
   zip -r my-custom-skill.zip .
   ```

4. 在 Claude.ai 设置中上传 ZIP 文件

## 💡 最佳实践

1. **开始简单**: 先使用 Anthropic 的预构建技能了解工作原理
2. **明确描述**: 编写清晰的技能描述，让 Claude 知道何时调用
3. **测试技能**: 上传自定义技能后，用不同提示词测试
4. **按目的组织**: 为不同目的创建独立技能，而不是一个"全能"技能
5. **版本控制**: 将自定义技能存放在 Git 仓库中管理
